@beekeeperstudio/sql-language-server

Demos of the SQL language server running in a Web Worker, driving CodeMirror 6.

SQLite + CodeMirror 6 demo

Real sql.js SQLite database, real schema queries via sqlite_master and PRAGMA table_info(), full autocomplete and diagnostics. Try typing SELECT * FROM and watching the table names appear, or SELECT bogus FROM users to see the unknown-column warning.

Raw LSP demo (no editor)

Just a textarea and "Get suggestions" / "Get diagnostics" buttons. See the exact LSP request and response, plus a log of every JSON-RPC message on the wire. Useful when something looks wrong in the editor demo — verifies the server in isolation.

The worker bundle is dist/worker.js. The host wires it to CodeMirror via a WorkerTransport that also answers the worker's bks/schema/* requests with a host-supplied SchemaProvider.
