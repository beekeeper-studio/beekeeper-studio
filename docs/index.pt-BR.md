## O que é o Beekeeper Studio?

O Beekeeper Studio é um gerenciador de banco de dados e interface SQL. É um aplicativo desktop que é fácil de usar. É uma alternativa visual para ferramentas de linha de comando como `psql` ou `mysql`, também fornecendo várias funcionalidades extras úteis.

### Plataformas Suportadas pelo Beekeeper Studio

O Beekeeper Studio está disponível para laptops e desktops executando [MacOS](./installation/macos-m1-intel.md), [Windows](./installation/windows.md) e [Linux](./installation/linux.md).

Importante, o Beekeeper Studio oferece **paridade completa de recursos** em todos os sistemas operacionais suportados.

Infelizmente, atualmente não temos planos de portar o Beekeeper Studio para dispositivos iOS ou Android.
