Vou ser sincero com você. Eu sou um desenvolvedor de software, você provavelmente também é. Eu amo tanto o software open source. Estou digitando isto em um laptop movido a Ubuntu usando o navegador Firefox. Ambos os softwares são open source, mas honestamente quase todo software que usamos é alimentado por open source de uma forma ou de outra.

O Beekeeper Studio não é diferente. Uma parte significativa do código-fonte do Beekeeper Studio está [disponível no GitHub](https://github.com/beekeeper-studio/beekeeper-studio) sob a licença GPLv3 (que ajuda a manter todos os forks open source também).

Como parte do meu compromisso com o open source, ofereço uma edição comunitária do Beekeeper Studio que é feita inteiramente de código open source. Ela tem menos recursos, mas para um estudante ou alguém que está apenas começando com bancos de dados, é perfeita.

Você pode baixar a edição comunitária em nossa [página de releases do GitHub](https://github.com/beekeeper-studio/beekeeper-studio/releases/latest).

## Disponibilidade de fonte dos componentes comerciais

Espero no futuro tornar todo o código do Beekeeper Studio disponível publicamente no GitHub, com o código comercial sendo [source available](https://en.wikipedia.org/wiki/Source-available_software).

## O Beekeeper Studio começou como um projeto open source

Devo o sucesso do Beekeeper Studio ao open source e à comunidade open source. Inicialmente toda a aplicação estava disponível sob a licença GPL.

À medida que o projeto cresceu, manter um projeto tão grande sem receita tornou-se impossível, então tomei a decisão de criar um fork comercial.

Mantenho uma edição comunitária generosa porque open source é muito importante para mim, e não quero deixar meus usuários gratuitos na mão. Se eu pudesse me sustentar com uma aplicação totalmente open source e gratuita, eu faria, mas infelizmente isso não é possível.

Todas as compras do Beekeeper Studio apoiam investimentos contínuos na edição comunitária open source.
