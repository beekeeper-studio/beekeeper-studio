## Simplemente descarga y ejecuta


Descarga y ejecuta el instalador de Windows desde [nuestro sitio web](https://beekeeperstudio.io).

Actualmente Beekeeper Studio solo es compatible con la arquitectura `x86_64`. El soporte para `ARM64` llegara en una version futura.


## Version portatil de Beekeeper Studio

Tambien tenemos una version portatil de Beekeeper Studio que no requiere instalacion. Esta tambien esta disponible en nuestro sitio web.
