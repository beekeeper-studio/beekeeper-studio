## Simplesmente Baixe e Execute


Baixe e execute o instalador Windows do [nosso site](https://beekeeperstudio.io).

Atualmente o Beekeeper Studio suporta apenas a arquitetura `x86_64`. Suporte para `ARM64` virá em uma versão futura.


## Versão Portátil do Beekeeper Studio

Também temos uma versão portátil do Beekeeper Studio que não requer instalação. Ela também está disponível no nosso site.
