## Baixe do Nosso Site
Para instalar o Beekeeper Studio baixe o arquivo instalador `dmg` do [nosso site](https://beekeeperstudio.io), então arraste o aplicativo Beekeeper Studio para sua gaveta de aplicativos.

Note que por padrão o MacOS irá prevenir você de instalar aplicativos de terceiros distribuídos (através da Mac app store). Para habilitar isso, navegue para `Configurações -> Segurança e Privacidade`, e habilite `App Store e Desenvolvedores Identificados`:

## Apple Silicon vs Intel

Instaladores do Beekeeper Studio estão disponíveis tanto para Intel quanto para Macs Apple silicon. Você pode escolher seu tipo de instalador preferido ao baixar do site.

!!! warning "Bibliotecas Oracle Database Não Suportam Apple Silicon"
    Bibliotecas e pacotes Oracle Database **não** estão disponíveis ainda para Apple Silicon. Então se você quer conectar o Beekeeper Studio ao seu Oracle Database você deve instalar a versão Intel do Beekeeper Studio e [executá-lo no Rosetta 2](https://support.apple.com/en-us/HT211861)


### Escolha Sua Arquitetura
Ao baixar o Beekeeper Studio, certifique-se de escolher a versão que corresponde à arquitetura do seu MacOS.
