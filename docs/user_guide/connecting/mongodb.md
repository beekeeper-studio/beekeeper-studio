# MongoDB Support

## Supported Features

- Table data view
- Table data sorting, filtering
- Table structure view
- Entity sidebar
- Editing data
- Running queries in some sort of REPL
- Writing SQL against Mongo
- Import/Export
- Backup/Restore
- Schema editing
- Read only mode
