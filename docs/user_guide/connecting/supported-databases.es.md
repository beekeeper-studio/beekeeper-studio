Beekeeper Studio tiene **paridad total de funciones** en todos los sistemas operativos compatibles, por lo que las bases de datos a continuacion funcionan igual en Windows, MacOS y Linux.

Para documentacion sobre como conectarte a tu primera base de datos, consulta la [guia de conexion a bases de datos](./connecting.md)

{% include-markdown "../../includes/supported_databases.md" %}

*[Soporte completo]: Totalmente compatible con el equipo de Beekeeper Studio y verificado como funcional.
*[Soporte Beta]: Totalmente compatible con el equipo de Beekeeper Studio, verificado como funcional, pero puede haber pequenos errores aqui y alla.
*[Proximamente]: Estamos trabajando en ello! Vuelve pronto para actualizaciones.
