Para visualizar o esquema do seu banco de dados, o Beekeeper Studio fornece Diagrama de Relacionamento de Entidades (ERD). Ele lhe dá uma visualização clara e interativa de como suas tabelas se conectam, para que você possa entender a estrutura de uma só olhada.

## Relacionamento Multi-esquema

![High-level ERD illustrating relationships between core user data and support ticket features.](../assets/images/diagrams-1.png)

O diagrama pode exibir relacionamentos que cruzam fronteiras de esquema. Tabelas de outros esquemas ainda aparecem na visualização, mas com uma distinção visual clara para que você sempre saiba quando um relacionamento se estende além do esquema ativo.

## Exportar como Imagem

![Export Diagram window showing a preview of the ER diagram, scale options, and buttons to copy or export the image.](../assets/images/diagrams-2.png)

Você pode salvar o diagrama atual como uma imagem ou copiá-lo para sua área de transferência. Isso torna fácil compartilhar o layout ou incluí-lo na documentação.

## Como Abrir o Diagrama ER

Você pode abrir o diagrama de **Tools > Open ER Diagram** para visualizar o banco de dados ativo.

![Tools menu showing the Open ER Diagram option.](../assets/images/diagrams-3.png)

Você também pode abri-lo **clicando com o botão direito em uma tabela ou esquema** na barra lateral.

![Right-clicking a table or schema in the sidebar showing the Open ER Diagram option.](../assets/images/diagrams-4.png)

Ou você pode abri-lo da **aba Structure** enquanto visualiza uma tabela específica. Há um botão de engrenagem na barra de status - clique nele, então selecione `View ERD`.

![Structure tab with the status-bar cog menu open showing View ERD.](../assets/images/diagrams-5.png)

## Preços do Diagrama ER

O Diagrama ER **está incluído em todas as versões pagas do Beekeeper Studio**, sem cobrança adicional.

Todos os downloads do Beekeeper Studio vêm com um teste gratuito de 14 dias, [baixe o Beekeeper Studio](https://beekeeperstudio.io/get) para experimentar o Diagrama ER agora mesmo.
